# 飞书分享插件测试文档

这是一个用于测试飞书分享插件的示例文档。

## 功能特性

- ✅ **一键分享**：快速将 Markdown 内容分享到飞书
- 🔐 **安全授权**：OAuth 2.0 标准授权流程
- 📁 **文件夹管理**：支持选择目标文件夹
- 🎯 **智能处理**：自动处理 Obsidian 特有语法
